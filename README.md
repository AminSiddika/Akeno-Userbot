### Deploy on VPS:
```console
@root: git clone https://github.com/TeamKillerX/Akeno-Userbot
@root: cd Akeno-Userbot
@root: pip3 install -r requirements.txt
@root: mv .env_sample .env
@root: nano .env
@root: python3 -m Akeno
```
